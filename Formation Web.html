Formation Développement Web

Nos programmes de formation bureautique s’adressent aux professionnels de tout niveau (public, débutant ou confirmé), dans le cadre du développement des compétences.

Cette formation vous permet d’apprendre des façons de créer les site web statique et dynamique.

Apprenez les langages fondamentaux pour développer des applications web (sites internet et applications) et construisez votre nouvelle carrière

Programme de la formation :

  • Maitriser les fondamentales de HTML
  • Maitriser les fondamentales de CSS
  • Maitriser les fondamentales de JAVASCRIPT
  • Maitriser les fondamentales de BOOTSTRAP
  • Maitriser les fondamentales de MySQL
  • Maitriser les fondamentales de PHP

Durée : 30 heurs
4 heures/semaine
Disponible en présentiel et en ligne
